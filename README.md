# Macine-Generated-Content-Detection-
Machine-Generated Content Detection using LSTM, BiLSTM, BERT, DistilBERT, and RoBERTa to classify AI-generated and human-written text. It uses the benchmark datasets like, RAID, MAGE, HC3+ and AI-DETECTION-PILE were used and the results were evaluated based on accuracy, precision, recall, and F1-score.
